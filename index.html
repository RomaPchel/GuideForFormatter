Installing JAVA

You need to install Java. It will help you to run the program on your device. Go to this site and chose preferred platform. I recommend you to download .exe installer as it will do everything automatically. The installation path should look similar to this (JDK version 18+).

Installing App

Press click! and download ZIP. Then Unzip archive in you preferred location. It should look something like this.

Go to folder "Out", scroll down and double click the file named QAFormatter.jar. Now you have everything to start. Good luck!

How to use

To add you file just simply drag and drop it.

To start formatting press "Format" and then press on you opened(!) word file.

You can easily mark your progress.

To see additional Information Hover over the text.

Pin\Unpin app so it will not close\hide when you work in Word.

Check hints to boost your performance.
